##1.0.0

 - 更新时间：2016年12月20日
 - 更新内容：规定Rating计算方式
	Rating计算方式：
	1. 一个题一分  
	
	2. 每次比赛第一名6分  
	
		 比赛第二名4分  
		 
		 比赛第三名3分  
		 
		 比赛其余名1分  
		 
	     同时比赛做出的题目依旧统计得分
	
	3. Rating积分名称颜色：  
	
		1000分以上红名  
		
		800-1000 橙色  
		
		600-800 紫色  
		
		400-600 蓝色  
		
		200-400 绿色  
		
		200以下灰色
